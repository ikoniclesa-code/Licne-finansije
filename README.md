# Lične finansije

Jednostavna aplikacija za praćenje ličnih finansija sa **dnevnim**, **mesečnim** i **godišnjim** nivoom pregleda.

## Šta aplikacija radi

- **Dodavanje transakcija** – prihod ili trošak, iznos, datum i opis (kategorija).
- **Dnevni nivo** – pregled transakcija za izabrani dan, sa zbirnim prihodom, troškom i bilansom.
- **Mesečni nivo** – pregled po mesecu (izbor mesec/godina), sa sumama i listom transakcija.
- **Godišnji nivo** – pregled po godini, sa sumama i listom transakcija.

Podaci se čuvaju u pregledaču (localStorage), tako da ostaju i posle zatvaranja stranice.

## Kako pokrenuti

1. Otvori folder u kome se nalazi `index.html`.
2. Dvostrukim klikom otvori `index.html` u browseru (Chrome, Edge, Firefox…),  
   **ili**
3. U Cursor-u: desni klik na `index.html` → *Open with Live Server* (ako imaš Live Server),  
   **ili** samo prevuci `index.html` u prozor pregledača.

Nije potreban server ni instalacija – sve radi iz jednog HTML fajla.
